# REGISTRATION-WIZARD-PRODESK-IT-SOLUTIONS

🚀 Registration Wizard

A production-grade multi-step registration form built using React, TypeScript, React Hook Form, and Zod.

The application demonstrates scalable form architecture, schema-based validation, reusable form components, and enterprise-level frontend engineering practices.

---

## 🚀 Features

✅ Multi-step registration flow

✅ Real-time form validation

✅ Schema validation using Zod

✅ React Hook Form integration

✅ Step persistence while navigating

✅ Dynamic progress bar

✅ Password show/hide toggle

✅ Review & submit step

✅ Responsive modern UI

✅ Accessible form inputs

✅ Smart validation handling

✅ Prevent invalid navigation

---

## 🧠 Tech Stack

- React.js
- TypeScript
- Vite
- React Hook Form
- Zod
- CSS3
